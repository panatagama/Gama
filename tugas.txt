Penentuan Tahun kabisat
1. masukan Tahun
2. cek apakah tahun kabisat atau bukan
    2.a Jika Tahun di bagi 4 = 0 dan Tahun di bagi 100 != 0 maka tampilkan 'Bukan Tahun Kabisat'
    2.b Jika Tahun di bagi 100 = 0 dan Tahun di bagi 400 = 0 maka tampilkan 3
3. Tahun Kabisat.

// pseudocode //
STORE 'Tahun' with any value number

IF 'Tahun' % 4 = 0 OR 'Tahun' % 100 = 0 AND 'Tahun' % 400 = 0
    DISPLAY 'Tahun Kabisat'
ELSE
    DISPLAY 'Bukan Tahun Kabisat'
END IF

// algoritma //
1. Masukan batasPakaian = 20
2. jumlah pakaian saat ini = 0
3. lakukan selama pakaian < batasPakaian
   3.a tambahkan pakaian 1
   3.b tampilkan pakaian
4. Nyalakan Mesin cuci.


// pseudocode //
STORE 'pakaian' with any value (20)
STORE 'counter' = 0

WHILE 'counter' less then 'pakaian' DO
    DISPLAY 'Masukan Pakaian'
    SET 'counter' = 'counter' + 1
    IF 'counter' = 20
        DISPLAY 'nyalakan mesin cuci'
    END IF
END WHILE

Seorang guru akan memeriksa kuku siswa-siswinya yang sebanyak 40 orang dengan cara berkeliling kelas. 
Jika guru menemukan siswa/siswi yang memiliki kuku yang panjang maka guru akan menghukum siswa/siswi 
tersebut, jika tidak guru akan memuji siswa/siswi tersebut.

// algoritma //
1. periksa kuku murid
2. batasMurid = 40
3. jumlah murid = 0
3. lakukan selama murid < batasMurid
    3.a jika kuku = panjang maka 'beri hukuman'
    3.b jika kuku = pendek maka 'beri pujian'
    3.c murid + 1
    3.d tampilkan 4
4. 'Tugas Selesai'

// pseudocode //
STORE 'murid' = 40
STORE 'counter' = 0

WHILE 'counter' less then 'murid' DO
    DISPLAY 'Periksa Murid Lainnya'
    SET 'counter' = 'counter' + 1
    STORE 'kuku' with any value
    IF 'kuku' = panjang
        DISPLAY 'beri hukuman'
    else
        DISPLAY 'beri pujian'
    END IF
END WHILE
